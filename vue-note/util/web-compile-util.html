# isUnaryTag

export const isUnaryTag = makeMap(
  'area,base,br,col,embed,frame,hr,img,input,isindex,keygen,' +
  'link,meta,param,source,track,wbr'
)

用makeMap方法生成一个判断是否是单标签元素。

# canBeLeftOpenTag

// Elements that you can, intentionally, leave open
// (and which close themselves)
export const canBeLeftOpenTag = makeMap(
  'colgroup,dd,dt,li,options,p,td,tfoot,th,thead,tr,source'
)

这个是判断能否自动补全的标签，这是浏览器的特性，比如说p标签'<p>我是p'，在浏览器中会自动补全为'<P>我是p</P>'。

# isNonPhrasingTag

export const isNonPhrasingTag = makeMap(
  'address,article,aside,base,blockquote,body,caption,col,colgroup,dd,' +
  'details,dialog,div,dl,dt,fieldset,figcaption,figure,footer,form,' +
  'h1,h2,h3,h4,h5,h6,head,header,hgroup,hr,html,legend,li,menuitem,meta,' +
  'optgroup,option,param,rp,rt,source,style,summary,tbody,td,tfoot,th,thead,' +
  'title,tr,track'
)

这个方法主要是收集非短语内容的标签，我们知道html都有它的内容模型，这个就是短语内容模型（Phrasing content）。
短语内容（Phrasing content）是流式内容的一个子集，定义了文本和它包含的标记，且可以在任何可以使用流式内容的地方出现。一些短语内容就构成了段落。
而p标签是属于流式内容，但是它又是特别的，它只能在里面放短语内容。
流式内容（Flow content）是一个广泛的类别，包括大多数可以包含在<body>元素之内的元素，包括标题元素、分段元素、措辞元素、嵌入元素、互动元素和表单相关元素。它还包括文本节点（但不包括那些只由空白字符组成的节点）。
所以上面这些标签都是不能放在p标签中的。
