# 结束感言

  文档到这里已经基本结束了，当初也不知道为什么会想要写文档。开始时间已经记不清了，可能是21年的某一天吧。一年多时间断断续续的，好多次想要放弃最后又默默的写了起来。在22年年后以第一天vue3.0正式发布，可我的2.0文档才刚刚起步，唯一的好处就是不会频繁更新了吧。感觉22年一年都过的浑浑噩噩的，只感觉到一年都在闹疫情。直到23年1月我发现git最早一个文件是22年1月21日提交的修改，我才意识到22年我什么都没有做。
  于是我开始拼命的去写，我从来都不知道我这么能抗，从早9点到晚12点，只要有空都在写文档。写到emo就休息一会儿继续写，从2月开始到3月近两个月时间天天如此，没有一天中断过，每天写到头昏脑胀才睡觉，睡前想的都是希望第二天早点到来我可以接着往下写，从来都没有这么渴望过明天。正文20w左右字，7w字是之前写的，我不到两个月产出居然是之前的近两倍。我都不知道自己是怎么坚持下来的，如果再让我写，我可能2w字都写不下去。但是我坚持下来了，虽然结果并不理想。
  我还有很多内容没有去写，可能以后会补，也可能不会。只能说基本告一段落了，这两天把文档都转成Markdown文档，两天时间处理了20w+字，肯定是没有仔细去整理，但是尽管这样我还是发现我很多笔误和阅读不畅的地方。但是恐怕我没有时间和精力去解决了，毕竟本质还是学习笔记，而不是教学文档。我希望别人可以看到它并能和我一起完善它。其实这种工作不应该由我一个人完成的，因为个人的思想总归是有局限的，不得不说一个人力量是有限的，而这种工作是需要灵感的，就算是我同一个人不同时间不同灵感对同一段代码产生的灵感都不同，但是很遗憾我没有找到Vue爱好者能和我一同完成它。
  说说我的收获吧，我不确定读源码带给我多大的收获，但是我能肯定是有很大收获的，因为从里面我每天都能看到我不懂知识，然后通过查找资料去学习它。再说说vue3.0去vue2.0的影响吧，可以确定的是vue3.0即使看起来与vue2.0相差很大，但其实整体设计思路不会有很大的变化，它不会过时。你可能会说vue3.0响应式原理都改变了，在我看来无非是换了一种API去实现数据劫持罢了，谈不上响应式原理改变。总之现在学习vue2.0也是有用的，我很确定，否则也不会坚持到现在。
  最后我对这个文档评价是：写的真的很差劲。但是感谢网上那些大神的文章和视频供我参考和学习（可能我不知道你们的名字）。当然最后还要感谢自己，感谢自己的坚持与努力！
